# BookShelf — Roadmap
    
    ## Current Phase
    
    Phase 0: Foundation — learning tools and concepts before writing project code.
    
    ## Planned Phases
    
    - Phase 0: Foundation (Git, HTTP, Spring concepts)
    - Phase 1: REST API (Book entity, CRUD, PostgreSQL)
    - Phase 2: Web UI (Thymeleaf)
    - Phase 3: Domain model (Authors, Tags, ManyToMany)
    - Phase 4: Validation, Logging, API docs
    - Phase 5: PDF upload and storage
    - Phase 6: Full-text search
    - Phase 7: Testing (JUnit, Mockito, Testcontainers)
    - Phase 8: E2E tests (Playwright)
    - Phase 9: Docker + Redis + Caching
    - Phase 10: Publisher release monitoring
    - Phase 11: CI/CD (GitHub Actions)
    - Phase 12: Deployment
